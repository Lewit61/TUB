# The Useful Box – File Storage

This repository contains the files used by **The Useful Box**.

The Useful Box is a collection of tools and resources that I use regularly. Instead of searching across different websites, find them all compiled here.

This repository acts as the **file storage backend** for the website and includes resources such as:

- Wallpapers
- Icons
- Utilities
- Other useful files

Each folder contains downloadable resources used on the website.

## Purpose

The goal of this repository is to provide a simple, organised location to host files used by **The Useful Box** website.

These files are intended to be accessed directly from the site for quick downloads and reference.

## Website

The Useful Box website provides a simple interface to browse and access these resources.

## Notes

- Files are organised for easy access and maintenance.
- New resources may be added over time.
- This repository is primarily for storage rather than development.
